Outgoing Adapter: AiClient

generateSeasonPlan(prompt)
  → client.prompt().user(prompt).call().content()
  → wrapped in Mono + boundedElastic scheduler
  → error handling + logging
Benefit: Switch from Gemini to OpenAI? Change only this class.

Incoming Adapter: AiResponseParser

parseAiResponse(raw, Type.class, fallback)
  → stripMarkdownFences(raw)
  → objectMapper.readValue(json, type)
  → fallback on parse failure
Benefit: Handles messy AI output (```json fences, whitespace) in one place.
The entire app is isolated from Gemini. Services never see ChatClient, Mono, or markdown fences.
