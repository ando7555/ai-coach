REST vs GraphQL — Getting Dashboard Data

✗ REST

GET /api/teams/1
GET /api/teams/1/players
GET /api/players/5/stats
GET /api/players/5/workload
GET /api/players/6/stats
GET /api/players/6/workload
...repeat per player
4+ round trips minimum

✓ GraphQL

POST /graphql

{ team(id: 1) {
  name, formation,
  players {
    name, rating, form,
    workload { fatigue }
  }
}}
1 request — exactly what you need
